How the column mapping works

The app fingerprints listing-manager exports by header pattern. When detected, bare Price and Date columns route to the listing fields — not the purchase fields — so listing prices aren't mistakenly imported as cost-of-goods.

// Common Questions

FAQ

Will it replace Nifty completely?

For inventory, sales, and the entire tax side — yes, completely. For the Poshmark sharing bot and 24/7 cloud automation — no. Honest answer: it depends on what you actually use Nifty for.

How does AI work without sending you my data?

The app calls the AI provider you choose with the API key you paste into Settings. There's no analytics, no phone-home, no auto-update server. You can also run locally with Ollama for $0 per listing.

One-time purchase — what about updates?

All updates are free for the life of the app. No maintenance fee, no renewal, nothing.

Can I use it outside the US?

Yes. Tax tools (Schedule C, 1099-K, quarterly) are US-specific, but everything else — inventory, sales, analytics, imports, exports, OCR, AI listings — works anywhere.

Refund policy?

14-day no-questions refund via Gumroad. Or try the free demo first.

I see multiple checkouts — does it matter which I use?

All checkouts deliver the same software and the same license. Gumroad is the default and fits most buyers. Lemon Squeezy is the best fit for EU / UK / AU buyers — they handle local tax inclusion so the price you see is the price you pay. Payhip is great if you found this through an affiliate link or prefer PayPal.

What if I share inventory with a partner?

Pro license includes 2 machine activations. The database file is portable — save it to Dropbox / OneDrive / iCloud and both computers can take turns editing it.

What platforms can I import sales from?

Pro auto-detects exports from Poshmark, eBay, Mercari, Depop, Etsy, Grailed, Whatnot, and Facebook Marketplace. Starter includes the first 3.

Does it support iOS/Android?

Desktop only — Windows, Mac, and Linux builds all ship in the same purchase (Linux as an AppImage that runs on any modern distro). The label print system supports printing to a phone via WiFi for in-app scanning, but the main UI is desktop.
