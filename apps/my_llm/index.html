// The Toolkit

A whole AI stack.
On one machine.

Hardware-aware model selection, a curated catalog, local-first routing with a budget, RAG, optional fine-tuning, a web console, and secure multi-user access — all stdlib-only.

🧭

Auto Hardware-Scaling

Probes CPU/RAM/GPU/VRAM, maps you to a tier, then picks the best model from your profile and shrinks the context to fit your VRAM. The right model, automatically.

📚

Catalog + Model Manager

A curated 2026 catalog — coding, general, reasoning, vision, fast, finance — with capability scores, sizes, and context. Browse your installed models, the full Ollama library, or Hugging Face — including the LM Studio, Jan/Cortex, Unsloth and Bartowski libraries (all just curated Hugging Face orgs) and the uncensored / “heretic” fine-tunes. Search by name, filter by capability and what your hardware runs, then add / edit / reorder in a click. No JSON.

🔀

Local-First Routing

Tries your local model first. Escalates to a Claude/OpenAI-compatible gateway only on error, low quality, high complexity, or a manual flag — never silently.

💰

Budget & Spend Cap

Daily and monthly USD caps on any paid escalation, plus a hard local_only offline switch. Every cent of cloud spend is logged. Local answers are always free.

🔌

OpenAI-Compatible /v1

A real /v1/chat/completions endpoint with /v1/models. Most OpenAI client code works by changing the base URL and key. Your apps consume your hub.

🖼️

Vision Pass-Through

Send standard OpenAI image parts; My_LLM routes to a local vision model and forwards the image. Photo → listing, OCR, chart Q&A — on your hardware. (Pro)

🔎

RAG Knowledge Base

The learn pipeline researches a topic on the web, builds a corpus, and indexes it for retrieval. Retrieved chunks can be injected into the local prompt before answering.

🎯

Optional QLoRA Fine-Tuning

Turn escalation logs into a training set and QLoRA/LoRA-tune the local model on the tasks it missed. Backend-aware: CUDA, ROCm, Apple MPS, or CPU.

🖥️

Setup Console

A localhost web console: guided wizard (system check, install, model download), dashboard, a Chat tab, and in-app Help docs. Run it as a browser tab or a chromeless app window.

👥

Secure Remote Multi-User

Serve friends and your other apps: per-user accounts, per-client API keys, RBAC (admin/user/pending), per-key budgets, allow/deny, and an access log — over a VPN or TLS. (Pro)

🛡️

Locked-Down By Default

Bound to 127.0.0.1 out of the box. A non-loopback bind refuses to start without accounts. Secrets live only in a host-locked auth.json, never served by the API.

📦

Single Binary, Zero Deps

Pure standard-library Python, compiled with Nuitka into one onefile executable. No Python install, no pip, no virtualenv. Runs from anywhere — even a USB stick.

🌐

Cross-Platform

Windows, macOS, and Linux. Inference acceleration via Ollama auto-detects CPU / NVIDIA CUDA / AMD ROCm / Apple Metal. llama.cpp is wired in as an alternative runtime.

// Common Questions

FAQ

Do I need a GPU?

No. The app runs on CPU and auto-scales the model to fit. A 16 GB CPU box runs a 7B model out of the box. If you have an NVIDIA, AMD, Intel, or Apple GPU, Ollama uses it automatically for faster inference.

Is it really free to run?

Local inference costs nothing per call — you already own the hardware. The only paid path is the optional cloud fallback, which uses your own API key and stays inside the daily/monthly USD cap you set.

What does "stdlib-only" mean for me?

The app itself has zero third-party Python dependencies. It compiles to one binary with no pip, no virtualenv, no DLL hell. Inference is handled by Ollama, which the wizard installs for you.

How do my other apps use it?

Create an API key in the console, point the app's base URL at http://<host>:8765/v1, and send the key. Most OpenAI client code works unchanged. A zero-dependency Python client is included.

One-time purchase — what about updates?

All updates are free for the life of the app. No maintenance fee, no renewal. Licenses never expire and are machine-locked (Starter = 1 machine, Pro = 2).

Is anything sent to you or the cloud?

No telemetry, no phone-home. Local prompts stay on your machine. The cloud fallback only fires when you've configured a key and a hard task crosses your threshold — and you can switch it off entirely with local_only.

Can friends and apps reach it safely?

Pro unlocks secure remote multi-user. The recommended setup is a VPN (Tailscale/WireGuard/OpenVPN) with My_LLM bound to the VPN interface — nothing is port-forwarded to the open internet. A TLS port-forward path is documented as an alternative.

I see multiple checkouts — does it matter which I use?

All checkouts deliver the same software and the same license. Gumroad is the default. Lemon Squeezy is the best fit for EU/UK/AU buyers (tax-inclusive pricing). Payhip is great if you prefer PayPal or arrived via an affiliate link.
